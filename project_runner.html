How to Run the Project

  1. Set up your environment:
    • Ensure you have AWS CLI, Python, and Boto3 installed.
    • Configure AWS CLI with your credentials using aws configure.
  2. Deploy CloudFormation Template:
    • Run the CloudFormation YAML file using the AWS Management Console or CLI.
  3. Run Backup to S3 Script:
    • Ensure the backup directory exists and is accessible.
    • Execute the backup_to_s3.sh script to create and upload backups to S3.
  4. Create RDS Instance:
    • Run the create_rds_instance.sh script to set up the database.
  5. Set up DNS Health Check:
    • Use the create_route53_health_check.sh script to configure health checks for failover.
  6. Automate Recovery:
    • Deploy the lambda_start_ec2.py script as a Lambda function.
    • Link the Lambda function to CloudWatch for automated triggers.

Refer to the individual script comments and documentation for further details on configuration and execution.
